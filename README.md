# The Weather App

<h2>Description :</h2>
This Project took a lot of hours to complete and was really difficult..

The Project Displays user's location's weather information by asking for its location information using the browser's geolocation API, if denied uses fetch api to get users location's information from geoapi and sends a fetch request to opemeteo api to get weather information and displays it.
